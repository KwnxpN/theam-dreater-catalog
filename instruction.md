# Project Setup (When You Already Have the Source Code)

If you do not have `pnpm` installed, install it first:

```bash
npm install -g pnpm
```

Then continue with the project setup steps below.

1. Open a terminal in the project root directory.
2. Install dependencies:

```bash
pnpm i
```

3. Start the development server:

```bash
pnpm dev
```

After running these commands, open the local URL shown in the terminal (usually `http://localhost:5173`).
